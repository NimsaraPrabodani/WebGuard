Tips for Identifying
Phishing Websites

  • Check the URL carefully for spelling mistakes.
  • Ensure the website uses HTTPS and has a valid certificate.
  • Look for poor design, broken links or grammar errors.
  • Verify the domain name before entering any information.
  • Be cautious of websites asking for sensitive information.
  • Hover over links to see the real destination.
  • Check for contact info and privacy policies.

Common Phishing
Techniques

  • Fake login pages that imitate trusted websites.
  • Emails or messages containing malicious links.
  • Urgent security alerts asking you to verify your account.
  • Fake banking or payment requests.
  • Prize, lottery, or gift card scams.
  • Social media impersonation attacks.
  • QR code phishing (Quishing).
  • Pop-up messages claiming your device is infected.

Safe Browsing
Guidelines

  • Keep your browser and operating system updated.
  • Use strong, unique passwords for every account.
  • Enable Two-Factor Authentication (2FA).
  • Avoid clicking suspicious links or downloading unknown files.
  • Use trusted antivirus and security software.
  • Only download software from official websites.
  • Regularly clear browser cache and cookies.

What to Do Before
Entering Passwords

  • Confirm the website address is correct.
  • Check for the HTTPS padlock icon.
  • Make sure the login page belongs to the official website.
  • Verify there are no spelling mistakes in the domain name.
  • Never enter passwords after clicking links from suspicious emails.
  • If unsure, manually type the website address into the browser.
  • Ensure the browser does not display security warnings.
  • Use a password manager to detect fake login pages.
